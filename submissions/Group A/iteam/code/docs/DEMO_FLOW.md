# VisionVoice Demo Flow (Webcam-Only “Glasses” Simulation)

## Goal
Run a reliable 60–90 second office demo that feels like smart glasses using only a webcam.

## Setup (1–2 min pre-demo)
- Webcam at eye level (clip to cap/headband if possible). If not, hold at eye level.
- Earbuds/headphones on the wearer.
- Props ready: 1 chair (hazard), 1 teammate “Angela” (pre-registered contact).
- Safety spotter walks beside the wearer to prevent collisions.

## Roles
- Presenter: narrates the story and keeps timing.
- Wearer: holds/wears webcam and speaks short voice prompts.
- “Angela”: steps into frame on cue.
- Safety spotter: quietly prevents bumps; does not narrate.

## 60–90 Second Run Of Show (Scripted)
### 0:00–0:10 — One-line framing
Presenter: “VisionVoice turns any webcam into smart-glasses-style guidance with short voice alerts.”

### 0:10–0:30 — Proactive hazard alert
Action: Place a chair in the wearer’s path; wearer walks slowly toward it.
Expected: “Chair ahead, slightly left.”
Action: Wearer sidesteps.

### 0:30–0:50 — Pre-registered person recognition
Presenter: “It only recognizes pre-registered contacts.”
Action: Angela steps into frame ~2–3 meters away, facing the camera.
Expected: “Angela is in front of you.”

### 0:50–1:10 — Reactive voice question
Wearer: “What’s in front of me?”
Expected: A short scene summary (obstacle/person/path direction).

### 1:10–1:25 — Caregiver escalation (optional if time is tight)
Wearer: “Call Angela” (or your configured safeword/command).
Expected: System initiates caregiver connection through Agora.

### 1:25–1:30 — Close
Presenter: “Any webcam, concise guidance, and human backup when needed.”

## Flow Checklist (Quick “Did We Show The Point?”)
- Hazard alert triggered before contact with the obstacle.
- Person recognition explicitly framed as “pre-registered only.”
- One voice Q&A demonstrates reactive guidance from live visual context.
- Escalation is demonstrated or verbally stated as a fallback if skipped.

## Reliability Moves (If Something Misbehaves)
- If hazard doesn’t trigger: slow down, fill more of the frame with the chair, re-approach.
- If person recognition fails: reduce backlight, move Angela closer, face camera, hold still briefly.
- If voice/Q&A is laggy: do one short question only (“What’s in front?”) and end the demo cleanly.
